# 💙💗 Digital Love Letter — JEEVANANDHAMVINODHINI

A personal digital love letter website built with HTML5, Vanilla JS, and Tailwind-inspired CSS.

## Hosting on GitHub Pages

1. Push this entire folder to a GitHub repository (e.g. `vino-letter`)
2. Go to **Settings → Pages**
3. Set **Source** to `main` branch, root folder `/`
4. Click **Save** — your live URL will be:
   ```
   https://<your-username>.github.io/vino-letter/
   ```

## File Structure
```
/
├── index.html              ← Main letter page
├── yes.html                ← YES response page
├── no.html                 ← NO response page
├── images/
│   ├── message page images/   ← 5 photos for main gallery
│   └── image for no/          ← 4 photos for NO page
└── README.md
```
